== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

After installed the theme, please read the documentation carefully before start using it.

Necessary plugins will be recommended to install via admin backend. You’d better to checkout the documentation for extra plugins that we used.

Recommended: if you want to customize the theme, you can use child theme and add your custom functions to child theme directory/functions.php.

For more information about child theme, read here: http://codex.wordpress.org/Child_Themes

If you have any questions related to the theme, please feel free to contact our support.
https://afthemes.com/supports/

= Does this theme support any plugins? =

ReviewNews includes support for the following plugins:
- Gutenberg: Supports a variety of block patterns for enhanced customization.
- Elementor: Compatible with template kits and custom widgets for advanced layouts.
- Beaver Builder: Compatible with templates and custom sections for advanced layouts.
- Brizy: Compatible with templates and custom sections for advanced layouts.
- WooCommerce: Fully compatible with eCommerce functionality, including product grids and custom shop layouts.
- Yoast SEO: Supports enhanced SEO features and breadcrumbs for improved visibility.
- Contact Form 7: Easily create contact forms and integrate them into your pages.
- Jetpack: Compatible with Infinite Scroll, Content Options, and Related Posts.
- WPML: Ready for multilingual websites, making translation and localization seamless.
